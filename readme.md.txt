Prueba #3: Listar Productos

**Objetivo:** Implementar una funcionalidad para listar y manejar productos.

**Descripción:**
- Usa el backend de las pruebas anteriores para recuperar una lista de productos y muéstrala en
una tabla en NextJS.
- La tabla debe ser interactiva, permitiendo ordenar por cualquier columna y buscar productos por
nombre.
- Implementa filtros avanzados para que los usuarios puedan buscar productos por rango de precio,
cantidad, etc.
- La tabla debe ser estilizada usando TailwindCSS para asegurar una buena experiencia de
usuario.
